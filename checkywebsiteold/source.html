    • Compile from source

    If you are the one who likes to compile software from source, follow the guide below. You can easily compile Checky from its source. The guide shows how to compile Checky from source on Windows and GNU/Linux systems. Checky was made in a MinGW and Dev C++ environment on Windows and G++ on Ubuntu.

    Windows Systems

    1. Download the source code from Github.You can even use Git. Type 'git clone https://github.com/Templatta/checky.git'
    2. Open the folder in your Command Prompt. Compile the program using MinGW. Type 'g++ main.cpp'.
    3. Congratulations, you have got a Checky executable to use!

    GNU/ Linux Systems

    1. Download the source code from Github.You can even use Git. Type 'git clone https://github.com/Templatta/checky.git'
    2. Install build essentials.
    3. Open the folder in your Terminal. Compile Checky using g++. Type 'g++ main.cpp'.
    4. Congratulations, you have got a Checky executable to use. Open the terminal and run it!




    Checky
    Checky is a free and open source tool that can make your life easier. It compares checksums and tells you the integrity status of your file. Comparing two checksums is the hardest thing to do. That is why we have created this tool for you. Don't bash your head. Just input the checksums and let Checky work its magic! Download now!
