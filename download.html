    • Download Checky for your system

    Checky is availible for download for free for Windows and GNU/Linux systems. Full support for Linux and MacOS will come soon. However, Checky can be run through WINE on GNU/Linux distros and MacOS. To build from source click here.



    Installation Instructions

    Windows Systems

    1. Open the .exe file and run it.
    2. Choose the desired type of install.(Administrator install will install Checky for all users.)
    3. Let Checky make changes to your system. Don't worry, we are not going to touch your files!
    4. Accept the licence, keep clicking next and let Checky install itself.
    5. Click on App icon on the desktop or Application Menu and enjoy using it! Easy as a pie!

    GNU/Linux Systems

    1. The full support for Linux systems is going to come soon. However, the application is functional enough.
    2. Download the checky.zip file by clicking on the above button. Extract it.
    3. Save checky.out it in the desired directory.
    4. Everytime you want to use Checky, open the directory in the terminal and type ./checky.out to initialize the application.
    5. Enjoy using Checky!
    The below example assumes that you have installed checky.zip in your home directory. Open the terminal.







    unzip checky.zip







    ./checky.out



    And you've done it!

    Checky
    Checky is a free and open source tool that can make your life easier. It compares checksums and tells you the integrity status of your file. Comparing two checksums is the hardest thing to do. That is why we have created this tool for you. Don't bash your head. Just input the checksums and let Checky work its magic! Download now!
