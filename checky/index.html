Get and compare the checksums of any file quickly

The installer works on Windows only


Checky allows you to get the checksum of a file and then compare it with the checksum given by the creator of the file to see if the file has been damaged or not. Checksums are long and hard to compare. Checky makes it easy by comparing the checksums for you!
It also stores history of checksums compared.

  • open source
  • free
  • lightweight
  • won't hog your memory

To use Checky, simply open up Checky by selecting the app from the Start Menu or searching it on your system. Checky’s interface will guide you on.

To compare checksums:
  • select 1 and press enter
  • enter file path
  • enter the type of checksum you want to compare
  • enter the checksum given by creator of the file. press enter and you will have the result ready

To access history, type 2 and press enter. To delete it, type 3 and press enter.


Checky is free and open source. It is distributed under the Checky license.

Checky’s source code can be accessed on Github here. It is written in Python 3 and uses a SQLite Database.
